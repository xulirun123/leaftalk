📝 使用说明

在 Vue 组件中使用:

// 在模板中
<div v-for="language in $supportedLanguages" :key="language.code">
  {{ language.name }}
</div>

// 切换语言
<button @click="$setLanguage('en-US')">Switch to English</button>

// 获取翻译
