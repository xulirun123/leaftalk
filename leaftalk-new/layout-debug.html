✅ 正确的布局方案

统一使用 MobileApp 的间距：

• MobileApp: padding-top: 65px (唯一的顶部间距)

• 页面级别: 不设置任何 margin-top 或 padding-top

• 内容区域: overflow-y: auto (可滚动)

• 第一项: 正常显示，不被遮挡

❌ 之前的错误方案

双重间距导致问题：

• MobileApp: padding-top: 65px

• 页面级别: margin-top: 65px 或 padding-top: 67px

• 结果: 总间距 130px+，第一项被推到很下面

• 问题: 滚动异常，内容被遮挡

🔧 修复要点

1. 统一间距策略：

只使用 MobileApp 的 padding-top: 65px，所有页面都不设置额外间距

2. 内容区域设置：

.content-area { flex: 1; overflow-y: auto; }

3. 第一项样式：

.current-location { margin: 0; flex-shrink: 0; }

4. 容器高度：

.region-selector { height: 100vh; }
