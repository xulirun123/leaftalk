路由配置：

{ path: '/settings/language-selector', name: 'LanguageSelector', component: () => import('../modules/settings/pages/LanguageSelector.vue'), meta: { title: '语言', requiresAuth: true } }

跳转逻辑：

// 通用页面点击语言 const handleLanguageClick = () => { router.push('/settings/language-selector') }

✨ 功能特性

  • ✅ 6种语言：简体中文、繁体中文、英语、日语、韩语、马来语
