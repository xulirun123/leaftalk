🛠️ 修复内容

✅ 已修复的问题

  • 替换自定义翻译函数：将 MobileProfile.vue 中的自定义 t 函数替换为 vue-i18n 的 useI18n
  • 添加完整翻译键：在所有 6 种语言文件中添加了 profile 相关翻译
  • 统一语言切换逻辑：确保 LanguageSelector 正确更新全局 locale
  • 响应式翻译：现在翻译会响应语言变化，无需刷新页面

🔧 技术修复详情

修复前（有问题的代码）：

// ❌ 自定义翻译函数，不响应语言变化 import { getCurrentInstance } from 'vue' const instance = getCurrentInstance() const t = (key: string) => instance?.appContext.config.globalProperties.$t(key) || key

修复后（正确的代码）：

// ✅ 使用 vue-i18n，响应语言变化 import { useI18n } from 'vue-i18n' const { t } = useI18n()
