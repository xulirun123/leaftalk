🔧 无限递归修复测试

✅ 修复完成 已解决

问题：选择语言时出现 "Maximum call stack size exceeded" 错误

原因：notifyLanguageChange 和 handleLanguageChange 之间形成无限递归

解决：分离公共方法和内部方法，添加重复检测机制

🚫 自动跳转已移除 用户体验改进

问题：选择语言后自动跳转到通用页面，用户无法继续选择

解决：移除2秒延迟的自动跳转，用户可以手动返回

🔴 修复前的问题

enableAutoTranslation() { // 触发事件 this.notifyLanguageChange() } handleLanguageChange() { // 又调用 enableAutoTranslation this.enableAutoTranslation() } // 结果：无限递归 💥

🟢 修复后的方案

enableAutoTranslation() { this.enableAutoTranslationInternal() this.notifyLanguageChange() // 触发事件 } enableAutoTranslationInternal() { // 内部逻辑，不触发事件 } handleLanguageChange(newLang) { if (this.currentLanguage === newLang) return this.enableAutoTranslationInternal() // 不触发事件 }
